# Security Policy

## Supported Versions

The following versions are currently supported with security updates:
- **1.0.x**

## Reporting a Vulnerability

We take the security of this project seriously. If you believe you've found a security vulnerability, please report it to us by creating a **Confidential Security Issue** on GitHub or by emailing the project maintainer at `a.issaoui@me.com`.

We will:
- Respond within 48 hours to acknowledge your report.
- Provide an estimated timeline for a fix.
- Notify you when the fix is released.

Thank you for helping keep this project secure!
