# MSDF Infinity Quality Demos 🎨

Welcome to the visual showcase for `universal-msdf-generator`. This directory contains interactive examples that demonstrate the power of **Multi-channel Signed Distance Fields** (MSDF).

## 🚀 PixiJS Infinity Zoom Demo

This demo showcases the "Infinity Quality" of our generated fonts. By using MSDF, we can scale text to massive sizes (up to 25.0x or more) while maintaining perfectly sharp, vector-like edges.

### Features
- **Orbitron Font**: A sharp, geometric font that highlights MSDF precision.
- **Interactive Zoom**: Range slider and mouse-wheel support for real-time inspection.
- **Ultra High-Performance**: Uses PixiJS v8's native, optimized MSDF rendering pipeline.

## 🛠️ How to Run

Because the demo loads assets (`.json` and `.png`) dynamically, it requires a local web server to avoid CORS (Cross-Origin Resource Sharing) restrictions.

### Option 1: Using the built-in server (No Dependencies)
We've provided a tiny, zero-dependency Node.js server for your convenience.

1. **Open your terminal** in the project root.
2. **Run the server**:
   ```bash
   node examples/serve.js
   ```
3. **Open the link**: [http://localhost:3003/](http://localhost:3003/)

### Option 2: Using your own server
If you have a global server tool installed, you can use that too:
```bash
npx serve .
# Then navigate to /examples/pixi-demo.html
```

## 📂 Asset Structure
- `/assets`: Contains the generated MSDF JSON layout and Atlas image.
- `pixi-demo.html`: The standalone visualization app.
- `serve.js`: The zero-dependency helper server.

## 🏗️ Generating more fonts for the demo
You can generate any font to test in the demo. For example, to test **Roboto Bold**:
```bash
npx tsx src/cli.ts "Roboto:700" --out examples/assets --format json
```
Then update the `fontFamily` in `pixi-demo.html` to `Roboto`.
